Speedmingle

2013 June

Speedmingle is a game I built with some friends at ThoughtWorks designed to be used at events (conferences, meetups etc). The purpose of the game was to get people socializing amongst each other and have a bit of fun. The stack we used was NodeJS and MongoDB on the server with client using web-sockets to make requests and update the scores in realtime. Those scores were then piped through D3.js to give some interesting visualiations. We used BurstSMS as our SMS API provider, and CircleCI and Heroku as part of our deployment pipeline.

Go Simple Ray

2014 February

I spent a few evenings in my spare time learning GoLang in preparation for a talk on "How to build a Raytracer" given at the monthly Go meetup. The raytracer is still relatively basic, only rendering spheres and planes, but it was designed and built as simply as possible, in an effort to help the talk. I wanted people to be able to perhaps use it as a point of reference when designing their own raytracing programs. I used JSON as a scene-description language allowing an easy way to serialize and deserialise scenes

A CHIP-8 emulator

2013 December

I spent a week of my personal time building an emulator designed to run 1970's era CHIP-8 games and programs. The emulator is built in Scala, but compiles to Javascript. I built the emulator as a learning exercises, and as a platform to test some functional programming theories I had learnt. I also built it with the intention of giving a lecture/talk on basic emulation theory for the ScalaSyd group, as it's very simple and easy to understand.
